TMV7

Tank Monitor App with Web Updater

TMV7 is an ESP8266-based water tank monitor. It reads level sensors and shows tank status in a simple local web interface.

The web updater lets you check for new firmware and install updates without reconnecting USB.

What It Does

  • Reads 4 level sensors (top to bottom).
  • Shows current tank percentage and sensor state.
  • Runs over Wi-Fi with captive-portal setup.
  • Provides a local browser UI for status and updates.

How Web Update Works

  • Device fetches manifest.txt from your server/GitHub.
  • If version_code is newer, update is offered.
  • Device downloads the linked .bin firmware file.
  • On success, device reboots into the new version.

OTA Files

Host these files in your repo (for example in ota/) and point firmware UPDATE_MANIFEST_URL to the manifest link.

Manifest URL Firmware URL

Update the firmware filename in the URL when you publish each new .bin.
